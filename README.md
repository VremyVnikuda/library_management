# Library Manager

## Описание
Приложение "Library Manager" — это консольное приложение для управления библиотекой книг.

## Функционал
1. **Добавление книги**: Пользователь вводит `title`, `author` и `year`. Книга добавляется с уникальным ID и статусом "в наличии".
2. **Удаление книги**: Удаление книги по её `ID`.
3. **Поиск книг**: Возможность искать книги по полям `title`, `author`, `year`.
4. **Отображение всех книг**: Вывод списка всех книг.
5. **Изменение статуса книги**: Изменение статуса книги на "в наличии" или "выдана".
6. **Сохранение данных**: Автоматическое сохранение данных в файл `books.json`.

1. Убедитесь, что Python установлен.
2. Запустите приложение:
   ```bash
   python library_manager.py
